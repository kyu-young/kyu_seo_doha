# 🐷 뚱순이의 엑셀 편집기

모니터링 리스트 엑셀 편집 & GPT 답변 관리 도구

## 🚀 바로 사용하기

### 웹에서 바로 열기:
**👉 https://kyu-young.github.io/kyu_seo_doha/doha_web_editor.html**

### 📱 iPhone/iPad에서:
1. Safari에서 위 URL 접속
2. 공유 버튼 → "홈 화면에 추가"
3. 앱처럼 사용!

## ✨ 주요 기능

- ✅ 엑셀 파일 열기/저장
- ✅ GPT 답변 편집 & 미리보기
- ✅ URL 일괄 관리
- ✅ 삭제 대기 목록 (구글 시트 연동)
- ✅ 프리셋 관리
- ✅ 다크모드 지원
- ✅ 오프라인 작동 (PWA)

## 🎨 2025 iOS 스타일 디자인

- Glassmorphism UI
- 완벽한 터치 최적화
- 부드러운 스크롤
- Safe Area 지원

## 💾 오프라인 지원
